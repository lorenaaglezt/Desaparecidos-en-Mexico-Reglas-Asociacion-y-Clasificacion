## Objetivo

Aplicar técnicas de minería de datos para obtener reglas de asociación que puedan brindar información sobre un conjunto de datos que contiene registros de personas desaparecidas en México. Además, aplicar modelos de clasificación que pueda brindar nuevas perspectivas al abordar esta problemática.
